# Movie_Recommender
🎬 An intelligent movie recommendation platform built using Machine Learning, NLP, and Streamlit that recommends similar movies using Content-Based Filtering and Cosine Similarity with an interactive UI.

# 🎬 CineMatch AI
### Intelligent Movie Discovery Platform

CineMatch AI is a Netflix-inspired movie recommendation system that helps users discover movies similar to their favorites using Machine Learning and Natural Language Processing (NLP).

The application analyzes movie metadata such as genres, keywords, cast, directors, and plot summaries to generate personalized recommendations and display them through an interactive Streamlit interface.

## 🧠 Machine Learning Workflow

1. Collect movie metadata
2. Clean and preprocess data
3. Create tags from movie features
4. Convert text into vectors using CountVectorizer
5. Calculate movie similarity using Cosine Similarity
6. Retrieve Top-N recommendations
7. Display recommendations through Streamlit
